## 根据ID选择
```html
<div id="myUniqueDiv">这是一个独特的div</div>

<script>
  // 获取元素
  const uniqueDiv = document.getElementById('myUniqueDiv');
</script>
```
## 根据Class选择
```html
<ul>
  <li class="list-item">项目 1</li>
  <li class="list-item">项目 2</li>
  <li class="list-item">项目 3</li>
</ul>

<script>
  // 获取所有class为list-item的元素
  const listItems = document.getElementsByClassName('list-item');
</script>
```
## 根据标签选择
```html
<p>这是段落1</p>
<p>这是段落2</p>

<script>
  const paragraphs = document.getElementsByTagName('p');
</script>
```
## 通过CSS选择器
```html
<ul>
  <li class="list-item">项目 1</li>
  <li class="list-item">项目 2</li>
  <li class="list-item">项目 3</li>
</ul>
<script>
const allListItems = document.querySelectorAll('.list-item');
</script>
```
### 基础选择器
| **选择器**    | **示例**          | **描述**                                     |
| ---------- | --------------- | ------------------------------------------ |
| **通用选择器**  | `*`             | 匹配 **所有** 元素。                              |
| **元素选择器**  | `p`             | 匹配 **所有** `<p>` 元素。                        |
| **ID 选择器** | `#header`       | 匹配 **ID 为 `header`** 的元素。在一个页面中 ID 应该是唯一的。 |
| **类选择器**   | `.box`          | 匹配 **class 包含 `box`** 的所有元素。               |
| **属性选择器**  | `[type="text"]` | 匹配 **拥有 `type` 属性且值为 `text`** 的所有元素。       |
### 组合选择器
| **选择器**     | **示例**    | **描述**                                        |
| ----------- | --------- | --------------------------------------------- |
| **后代选择器**   | `div p`   | 匹配 **所有作为 `div` 元素后代** (不论层级深浅) 的 `<p>` 元素。   |
| **子选择器**    | `div > p` | 匹配 **所有作为 `div` 元素直属子元素** 的 `<p>` 元素。         |
| **相邻兄弟选择器** | `h1 + p`  | 匹配 **紧跟在 `h1` 元素后面** 的 **第一个** `<p>` 元素。      |
| **通用兄弟选择器** | `h1 ~ p`  | 匹配 **跟在 `h1` 元素后面** 的 **所有** `<p>` 元素 (不必紧邻)。 |
### 结构性伪类
| **伪类**               | **描述**                                                              | **示例用法**                                       |
| -------------------- | ------------------------------------------------------------------- | ---------------------------------------------- |
| `:first-child`       | 匹配父元素的**第一个子元素**。                                                   | `p:first-child { font-weight: bold; }`         |
| `:last-child`        | 匹配父元素的**最后一个子元素**。                                                  | `li:last-child { margin-bottom: 0; }`          |
| `:only-child`        | 匹配**既是第一个子元素又是最后一个子元素**（即它是其父元素下的唯一子元素）。                            | `div > p:only-child { font-style: italic; }`   |
| `:nth-child(n)`      | 匹配父元素的**第 $n$ 个子元素**。$n$ 可以是数字、关键字（如 `odd` 或 `even`）或表达式（如 `2n+1`）。 | `li:nth-child(2n)` 匹配偶数行的 `<li>`。              |
| `:nth-last-child(n)` | 匹配父元素的子元素中**倒数第 $n$ 个**。                                            | `li:nth-last-child(1)` 匹配倒数第一个 `<li>`。         |
| `:first-of-type`     | 匹配父元素中**该类型元素**的第一个。                                                | `p:first-of-type` 匹配父元素中**第一个** `<p>` 元素。      |
| `:only-of-type`      | 匹配父元素中**该类型元素**的唯一一个。                                               | `span:only-of-type` 匹配父元素中**唯一**的 `<span>` 元素。 |
### 交互状态伪类
| **伪类**     | **描述**                        | **示例用法**                                        |
| ---------- | ----------------------------- | ----------------------------------------------- |
| `:hover`   | 当鼠标指针悬停在元素上时。                 | `a:hover { color: red; }`                       |
| `:active`  | 当元素被用户激活（如鼠标按下或按键按下）时。        | `button:active { background-color: darkblue; }` |
| `:focus`   | 当元素获得焦点时（常用于表单元素如 `<input>`）。 | `input:focus { border: 2px solid blue; }`       |
| `:link`    | 匹配尚未访问的 `<a>` 元素。             | `a:link { color: blue; }`                       |
| `:visited` | 匹配已被用户访问过的 `<a>` 元素。          | `a:visited { color: purple; }`                  |
### 表单状态伪类
| **伪类**      | **描述**                                               | **示例用法**                                                   |
| ----------- | ---------------------------------------------------- | ---------------------------------------------------------- |
| `:enabled`  | 匹配所有可用的（启用状态的）表单元素。                                  | `input:enabled { opacity: 1; }`                            |
| `:disabled` | 匹配所有被禁用的表单元素。                                        | `input:disabled { background-color: #eee; }`               |
| `:checked`  | 匹配已被选中的**复选框**（checkbox）或**单选框**（radio）。             | `input[type="checkbox"]:checked + label { color: green; }` |
| `:valid`    | 匹配内容通过验证（如 `<input type="email">` 中输入了有效的邮箱格式）的表单元素。 | `input:valid { border-color: green; }`                     |
| `:invalid`  | 匹配内容未通过验证的表单元素。                                      | `input:invalid { border-color: red; }`                     |
### 其他伪类
|**伪类**|**描述**|**示例用法**|
|---|---|---|
|`:not(selector)`|匹配**不符合**括号内选择器的元素。|`p:not(.intro)` 匹配所有**不具有** `intro` 类的 `<p>`。|
|`:empty`|匹配**没有**任何子元素的元素（包括文本节点和注释）。|`div:empty { display: none; }`|
### 伪元素
|**伪元素**|**示例**|**描述**|
|---|---|---|
|`::before`|`p::before`|在元素**内容**的**前面**插入生成的内容。|
|`::after`|`p::after`|在元素**内容**的**后面**插入生成的内容。|
|`::first-line`|`p::first-line`|匹配元素中的**第一行文本**。|
|`::first-letter`|`p::first-letter`|匹配元素中的**第一个字母**。|
|`::selection`|`::selection`|匹配用户**选择**（高亮显示）的部分。|
