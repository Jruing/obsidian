```bash
uv tool install crewai
```
## 生成项目框架
```bash
crewai create crew <your_project_name>
```
## 核心功能文件列表
| 文件            | 目的             |
| ------------- | -------------- |
| `agents.yaml` | 定义你的 AI 代理及其角色 |
| `tasks.yaml`  | 设置代理任务和工作流程    |
| `.env`        | 存储 API 密钥和环境变量 |
| `main.py`     | 项目入口点与执行流程     |
| `crew.py`     | 船员编排与协调        |
| `tools/`      | 自定义代理工具目录      |
| `knowledge/`  | 知识库目录          |
## 运行项目
```bash
crewai run
```
