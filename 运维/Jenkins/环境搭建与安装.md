## docker安装
### 拉取镜像
```
docker pull jenkins/jenkins:latest
```
### 创建Jenkins工作目录
```
mkdir -p /data/jenkins_home
chmod R 777 /data/jenkins_home
```
### 启动Jenkins
```
# 启动Jenkins容器
docker run -d   --name jenkins   --restart always   -p 8087:8080  -p 50000:50000   -v /data/jenkins_home:/var/jenkins_home    -v /var/run/docker.sock:/var/run/docker.sock -e TZ="Asia/Shanghai"  jenkins:latest
# 8087 Web管理界面端口
# 50000 代理节点通信端口
```
### 访问
http://127.0.0.1:8087
### 获取密码
```
cat /data/jenkins_home/secrets/initialAdminPassword
```
