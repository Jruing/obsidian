## 创建/编辑文件
```bash
nano a.txt
```
## 文件操作与退出
| **快捷键**          | **底部显示**       | **功能描述**                               |
| ---------------- | -------------- | -------------------------------------- |
| **`Ctrl` + `X`** | `^X Exit`      | **退出** Nano。如果文件未保存，会提示你是否保存。          |
| **`Ctrl` + `O`** | `^O WriteOut`  | **保存** 当前文件（写入磁盘）。                     |
| **`Ctrl` + `R`** | `^R Read File` | **插入** 另一个文件的内容到当前光标位置。                |
| **`Ctrl` + `S`** | `^S Save`      | 另存为（在某些较新的版本中替代 `Ctrl` + `O` 或用于创建备份）。 |
| Alt + U          | M-U            | 撤销上一步操作                                |
## 搜索与替换
| **快捷键**          | **底部显示**      | **功能描述**                       |
| ---------------- | ------------- | ------------------------------ |
| **`Ctrl` + `W`** | `^W Where Is` | **搜索** 文本。                     |
| **`Alt` + `W`**  | `M-W Next`    | 在搜索结果中跳转到 **下一个** 匹配项（重复上次搜索）。 |
| **`Ctrl` + `\`** | `^\ Replace`  | **替换** 文本（先输入要搜索的词，再输入要替换成的词）。 |
| Alt + R          | M-R           | 开启 / 关闭正则表达式搜索                 |
## 复制/剪切/粘贴
| **快捷键**          | **底部显示**   | **功能描述**                     |
| ---------------- | ---------- | ---------------------------- |
| **`Ctrl` + `K`** | `^K Cut`   | **剪切** 当前行。如果先选中了文本，则剪切选中的块。 |
| **`Ctrl` + `U`** | `^U UnCut` | **粘贴** 剪切缓冲区中的内容到光标位置。       |
| **`Alt` + `^`**  | `M-^ Copy` | **复制** 当前行。如果先选中了文本，则复制选中的块。 |
| **`Ctrl` + `6`** | `^6 Mark`  | 启动/取消 **标记** 文本（开始/结束选择）。    |
## 光标跳转
| **快捷键**          | **底部显示**         | **功能描述**                       |
| ---------------- | ---------------- | ------------------------------ |
| **`Ctrl` + `Y`** | `^Y Prev Page`   | **上翻页**（向前滚动一页）。               |
| **`Ctrl` + `V`** | `^V Next Page`   | **下翻页**（向后滚动一页）。               |
| **`Ctrl` + `A`** | `^A Go to Begin` | 移动光标到 **行首**。                  |
| **`Ctrl` + `E`** | `^E Go to End`   | 移动光标到 **行尾**。                  |
| **`Ctrl` + `_`** | `^L Go To Line`  | **跳转** 到指定的行号和列号（例如输入 `10,5`）。 |
| Alt + \          | M-\              | 光标移到文件开头                       |
| Alt + /          | M-/              | 光标移到文件结尾                       |
| Alt + G          | M-G              | 跳转到指定行号（输入行号回车）                |
## 其他快捷键
| **快捷键**          | **底部显示**       | **功能描述**                                  |
| ---------------- | -------------- | ----------------------------------------- |
| **`Ctrl` + `C`** | `^C Cur Pos`   | **显示** 当前光标所在的 **行号、列号** 和文件百分比。          |
| **`Ctrl` + `G`** | `^G Get Help`  | 打开 **帮助** 界面，查看所有快捷键。                     |
| **`Ctrl` + `J`** | `^J Justify`   | **对齐** 当前段落（重新格式化）。                       |
| **`Ctrl` + `T`** | `^T To Spell`  | **拼写检查**（如果安装了拼写检查工具）。                    |
| **`Alt` + `A`**  | `M-A Mark All` | 启用/禁用 **智能家居高亮**（Smart Home Key, 用于快速移动）。 |
