## 安装goctl
```
go install github.com/zeromicro/go-zero/tools/goctl@latest
```
## 安装protoc
```
goctl env check --install --verbose --force
```
## 安装go-zero
```
go get -u github.com/zeromicro/go-zero@latest
```
## 创建项目
```
mkdir projectName
cd projectName
go mod init projectName
```
